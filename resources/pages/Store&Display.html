Store and Display

This project was a brief give me to create a storage unit for the "Little Black Classics box" of Penguin books. My main priority for this project was to ensure all books can be stored and displayed as this was the main focus of the brief. I followed this process by creating designs of a unique bookshelf. This bookshelf is design to fit both in a library setting but also in the home. This product can also go back to back to create an island in the space. (penguin island) I imagined this in a library as a place to incorage children to read due to the reachable size and interesting design to intrigue them.

Panelled pine sheets with pine frame thus making mass production an option.
